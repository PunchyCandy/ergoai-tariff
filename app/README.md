# ErgoAI Tariff Application

This app now supports two modes:

- the original facts-only Ergo demo in [`main.ergo`](./main.ergo)
- a real Python CLI in [`main.py`](./main.py) that accepts shipment inputs and returns computed duty results

## What The CLI Does

`main.py` loads the Ergo knowledge base from:

- `data/products.ergo`
- `data/shipments.ergo`
- `policy/usa_base.ergo`
- `policy/usa_exec_overrides.ergo`
- `policy/exemptions.ergo`
- `rules/decision.ergo`

It then either:

- evaluates an existing sample shipment id
- evaluates a new shipment you provide at runtime
- or prompts you interactively in the terminal

## Requirements

You need ErgoAI available through:

- `ERGOROOT`
- `XSBARCHDIR`

Those can be supplied through your shell environment or a `.env` file.

## Commands

List the sample shipments already in the knowledge base:

```bash
python3 app/main.py list
```

Evaluate a sample shipment:

```bash
python3 app/main.py sample s1
```

Evaluate a runtime shipment using an existing product id:

```bash
python3 app/main.py quote \
  --origin china \
  --import-country usa \
  --product-id p_router \
  --value 1000 \
  --date 2025-11-10
```

Evaluate a runtime shipment with a custom HS code:

```bash
python3 app/main.py quote \
  --origin china \
  --import-country usa \
  --product-id custom_router \
  --hs-code hs8517_62 \
  --category electronics \
  --description wireless_router \
  --value 1000 \
  --date 2025-11-10
```

Run in interactive mode:

```bash
python3 app/main.py interactive
```

Return machine-readable JSON only:

```bash
python3 app/main.py quote \
  --origin japan \
  --import-country usa \
  --product-id p_parts \
  --value 1200 \
  --date 2026-02-02 \
  --json
```

## Output Shape

The CLI returns structured output with:

- `input`: the shipment facts used for reasoning
- `product`: the resolved product metadata
- `result`: the computed duty, rate, and Ergo explanation

## Notes

- `app/tmp_combined_rules.ergo` is generated by `main.py` before querying Ergo.
- The facts-only demo remains useful if you want a guaranteed-simple Ergo load path.
